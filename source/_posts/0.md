0x0000 0001首先他是个16进制的数字、

8进制的是0开头的、比如 077 他是八进制的、十进制的话就是63、7*8+7=63

0x0000 0001他表示一个32位的、

因为十六进制的一位有16种变化、四位的变化也是16种。

那么、想表示32位的数据、需要16进制的

```c
bit  0000 0000 0000 0000 0000 0000 0000 0001
0x    0    0    0    0    0    0    0    1
```
